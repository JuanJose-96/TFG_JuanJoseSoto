## Aplicación FastFix (Gestión de clientes y técnicos)
### Descripción
La aplicación busca ser el intermediario y puente que conecta a clientes con técnicos cualificados de distintos
sectores (electricistas, carpinteros, etc.) para resolver servicios de forma ágil y profesional.
Permite buscar y filtrar profesionales por ciudad, sector y puntuacion de reseñas, facilitando la elección del técnico
más adecuado.
